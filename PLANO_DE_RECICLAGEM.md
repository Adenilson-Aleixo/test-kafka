# Plano de Reciclagem – Desenvolvedor Kotlin/JVM Sênior

Duração sugerida: **12 semanas (3 meses)**

Objetivo: atualizar conhecimentos de Kotlin moderno, arquitetura, performance, cloud e observabilidade.

---

# Fase 1 – Kotlin Moderno (Semana 1–2)

## Tópicos para estudar

- [ ] Coroutines avançadas
- [ ] Flow
- [ ] Structured Concurrency
- [ ] Inline classes
- [ ] Extension functions avançadas
- [ ] DSLs em Kotlin
- [ ] Imutabilidade e design funcional

## Prática

Criar um pequeno serviço:

```
order-service
```

Utilizando:

- [ ] Coroutines
- [ ] Flow
- [ ] Data classes
- [ ] Testes unitários

# Fase 2 – Spring Boot com Kotlin (Semana 3–4)

## Estudar

- [ ] Spring Boot 3 com Kotlin
- [ ] Spring Data
- [ ] Spring Security com JWT
- [ ] Resilience patterns (retry, circuit breaker)
- [ ] Observabilidade

## Projeto

```
payment-api
```

Com:

- [ ] autenticação JWT

- [ ] tratamento de falhas
